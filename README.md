# Gramn!

grammar checker cli for Saplin

## Supported languages:

 - pt-br

## How?

 - Saplin key needed.
 - Create a `.env` file.
 - `./gramn <text_file>`

## Dictionary files:

  Put in the same directory as the script.
  - `user_dict` contains errors to ignore.
  - `ignore_alts` ignores html alt tag if == file line.
